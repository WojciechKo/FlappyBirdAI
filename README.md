GosuFlappy
==========

GosuFlappy is just a poor copy of FlappyBirds build with ruby 2.0 and Gosu Gem (https://github.com/jlnr/gosu)

##Installation

  - Clone the repository.
  - Install ruby 2.0
  - Install Gosu Gem
```sh
  gem install gosu
```
###Launch it
```sh
  ruby flappy.rb
```
